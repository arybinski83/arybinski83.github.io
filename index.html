Programming Projects

A collection of projects resulting in various programs ranging from calculators to weather lookup.

Word Count: program for looking up whole words in a body of text and classifying them by the number of times the word was used.

Calculator: basic program that can be used as a calculator for day-to-day calculations but is limited to 5 functions: Add, Subtract, Multiply, Divide, and Average

Order & Receipt: program that can be used to determine the cost of an item based on the amount ordered defined by predetermined unit process and generate an invoice/receipt for the order.

Cash Register: program that is used to create a receipt for a purchase based on user inputs of price per unit. The receipt generates a total as well as a count of items purchased.

Weather Lookup: program with API connectivity designed as a weather lookup tool based on multiple inputs such as city or zip code. The information provided is live and based on a weather service.
